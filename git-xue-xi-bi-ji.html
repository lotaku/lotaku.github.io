git 学习笔记

问题记录及解决方法

  • 在git clone或者git add remote {server} 时，报错，说证书校验有问题

    报错代码

    error: server certificate verification failed. CAfile: /etc/ssl/certs/ca-certificates.crt CRLfile: none

最简单的解决方法是调用前加一个环境变量：

$ export GIT_SSL_NO_VERIFY=1

但是这个方法肯定不方便，更好的方法是：

$ git config --global http.sslverify false

参考：http://blog.csdn.net/ubuntu64fan/article/details/12373403
